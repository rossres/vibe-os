# Step 9: Campaigns

## MANDATORY EXECUTION RULES
- NEVER create paid campaigns without explicit budget approval
- ALWAYS link content to campaigns before activating
- Follow the stage playbook — don't skip the funnel

## DEPENDENCIES
- Step 5 (System Charter) — budget and vertical allocation
- Step 8 (Content Engine) — need content to attach to campaigns

## YOUR TASK
Set up the campaign structure — platforms, ad groups, targeting, and content links. The campaign playbook maps awareness stages to recommended plays.

## EXECUTION SEQUENCE

### Phase 1: Platform Selection

Ask: "Which advertising platforms do you want to use?"
- Google Ads (search intent — highest conversion)
- Meta Ads (awareness + retargeting)
- LinkedIn Ads (B2B targeting)
- Email (outbound sequences)
- Organic social (LinkedIn, Instagram)

### Phase 2: Stage Playbook Review

Present the built-in playbook for each stage:

```
STAGE PLAYBOOK:

identified → Google Search ads (high intent), cold email
aware → Retargeting, social proof content
interested → Case studies, comparison content, email nurture
consider → Demo/trial offers, landing pages, direct outreach
selecting → Closing offers, urgency, 1:1 outreach
```

Ask: "Does this match your sales process? Want to adjust?"

### Phase 3: Campaign Creation

For each active platform + vertical combination:
1. Create the campaign record
2. Set budget (from charter allocation)
3. Create ad groups with targeting
4. Link content from Step 8
5. Set status to "draft" (not active yet)

### Phase 4: Review & Activate

Present all draft campaigns for review:
- Platform, budget, targeting, linked content
- User approves → status changes to "active"
- User defers → remains in "draft"

### Phase 5: Document

Append to session document:
- Campaign structure (platforms, budgets, targeting)
- Content linked to each campaign
- Status (active vs draft)

## OUTPUTS
- Campaign records in database
- Ad groups and content links
- Session document updated

## SUCCESS METRICS
- At least 1 campaign per Tier A vertical
- Budget aligns with charter
- Content is linked (no empty campaigns)
- User understands the stage playbook
