# Step 4: Positioning

## MANDATORY EXECUTION RULES
- "Only we" statements must be LITERALLY TRUE — do not invent claims
- NEVER generate competitive positioning without user validation
- The comparison table must include the user's actual competitors

## DEPENDENCIES
- Step 2 (Brand Brain) — uses Narrative Framework and canonical narrative
- Step 3 (ICP & Personas) — uses vertical context for positioning

## YOUR TASK
Define competitive positioning: only-we statements, key differentiators, competitor comparison, and pricing narrative. This drives all ad copy, battle cards, and objection handling.

## EXECUTION SEQUENCE

### Phase 1: Only-We Statements

Ask: "What can you truthfully say that NO competitor can?"

Guide the user through 5-7 statements. Each must be:
- Literally true (not aspirational)
- Specific (not "best customer service")
- Defensible (they could say it in a courtroom)

### Phase 2: Competitor Comparison

1. "Who are your top 3-5 competitors or alternatives?"
2. For each competitor, discuss: cost, setup time, availability, customer experience, conversion rate
3. Build a comparison table that's honest — don't strawman competitors

### Phase 3: Key Differentiators

Distill 5-7 differentiators from the only-we statements and comparison. These are the talking points that appear in all content.

### Phase 4: Pricing Narrative

1. "What's your pricing model?" — tiers, features, pricing
2. "How do you frame ROI?" — cost vs. cost of inaction
3. "What's the per-unit economics?" — cost per interaction, cost per missed opportunity
4. "How do you compare on price?" — vs. alternatives

Write pricing config.

### Phase 5: Stage Messaging

For each awareness stage (identified → selecting), define:
- Headline template (with {vertical} and {revenue} placeholders)
- CTA text

### Phase 6: Write Config

Update `config/brand.yaml` with:
- competitive section
- pricing section
- stage_messaging section

## OUTPUTS
- `config/brand.yaml` — updated with competitive, pricing, and stage messaging
- Session document updated with positioning summary

## SUCCESS METRICS
- 5+ only-we statements that are literally true
- Comparison table with real competitors
- Pricing tiers with ROI framing
- Stage messaging with escalating urgency
- User confirms all claims are truthful
